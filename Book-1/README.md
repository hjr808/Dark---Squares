## Thematic Framework

### Core Concept: Dark Squares
Every chapter should reinforce the central metaphor:
- **Choices** create shadows
- **Shadows** become consequences
- **Consequences** define character
- **Moral weight** drives narrative

### Established Taglines (use as tone reference)
- "Every generation receives an omen"
- "The streets, the empire, the oath"

## Version Control

### File Naming
- **Main manuscript:** `Dark-Squares-Book-1.md`
- **Versions:** `Dark-Squares-Book-1_Draft-1.md`, etc. (if archiving)
- **Status in name:** `_DRAFT`, `_REVISION`, `_FINAL`

### Revision Process
1. Draft complete in main file
2. Peer review by 2-3 team members
3. Incorporate feedback
4. Archive previous version (if major revision)
5. Commit with clear message
