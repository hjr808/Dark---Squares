## Character Profile Template

Each character file includes:

### Basic Information
- Full name, aliases, nickname
- Age range
- Background/origin

### Physical Description
- Height, build, distinguishing features
- Clothing style and aesthetic
- Visual reference (from Producer Edition posters)

### Personality & Voice
- Core personality traits
- Speech patterns and dialect
- Emotional baseline
- Motivations and desires

### Character Arc
- **Book One status:** Starting point
- **Key transformation moments:** Major scenes that change character
- **Book Two trajectory:** Evolution continues
- **Book Three endpoint:** Where character ends (if applicable)

### Relationships
- Primary relationships with other characters
- How character is perceived by others
- Conflict points and allies

### Key Scenes
- Defining moments for this character
- Scenes showcasing character's core conflict
- References to specific chapters/episodes

## Character Development Process

### Initial Creation
1. Define core concept and archetype
2. Establish basic information and voice
3. Create visual reference (from Producer Edition)
4. Write initial character profile

### Development During Writing
1. Reference profile during scene writing
2. Note any emergent character traits
3. Track relationship changes
4. Document key scenes and moments

### Revision & Refinement
1. Review character arc for consistency
2. Update profile after major scenes written
3. Cross-check with other characters' perspectives
4. Finalize for production use

---

## Character Consistency Guidelines

### Voice Consistency
- Each character has distinct speech patterns
- Vocabulary reflects background and education
- Emotional expression is consistent with personality
- Character growth doesn't mean personality change

### Relationship Dynamics
- Character's view of others may differ from reality
- Relationships evolve but core dynamics remain
- Conflicts rooted in character motivations
- Alliances based on character values

### Physical Consistency
- Visual appearance matches character profile
- Clothing choices reflect personality and circumstances
- Physical mannerisms consistent throughout
- Visible changes mark significant character moments
