## Episode Structure

### Standard TV Episode Format
```
Teaser (5-7 pages)
Act One (10-12 pages)
Act Two (10-12 pages)
Act Three (10-12 pages)
Act Four (8-10 pages)

Total: ~45-50 pages (1 page ≈ 1 minute screen time)
```

### Episode Components
- **Cold open/Teaser:** Hook audience, establish tone
- **Acts:** Divided by commercial breaks (TV format)
- **Cliffhanger/Resolution:** Lead to next episode or conclude arc
- **Tag:** Optional short scene after act break

## Screenplay Writing Standards

### Format
- **Software:** Final Draft, ScreenFlow, or Celtx
- **Standard Format:** Industry-standard screenplay format
- **Font:** Courier, 12pt
- **Margins:** Standard TV screenplay margins

### Character Names
- **First Appearance:** ALL CAPS
- **Subsequent:** Normal formatting
- **Consistency:** Use exact character names as established

### Scene Headings
```
INT. MANNIE'S APARTMENT - NIGHT
EXT. STREET CORNER - DAY
```

### Action Lines
- Clear, visual descriptions
- Active voice preferred
- Avoid camera directions (except when essential)
- Details that matter to story

### Dialogue
- Authentic to character
- Subtext crucial
- Realistic speech patterns
- Regional authenticity

---

## Content Guidelines

### Tone & Style
- Urban realism with dramatic weight
- Consequence-focused storytelling
- Moral complexity without clear "good" or "bad"
- Character-driven narrative
- Thematic consistency ("dark squares" concept)

### Visual References
- Use Brand Guidelines for visual tone
- Reference Producer Edition materials for look/feel
- Describe scenes to match established aesthetic
- Lighting and color grading consistent with series

### Character Consistency
- Reference character profiles for voice/motivation
- Maintain established relationships
- Character development across episodes
- Consistent with Book One narrative

## Writing Process

### Before Writing
1. Confirm episode outline
2. Review character profiles
3. Check book manuscript content for episode
4. Verify location/setting details
5. Establish emotional beats for episode

### During Writing
1. Write in industry-standard format
2. Reference character voices
3. Maintain series tone
4. Track scene count and page length
5. Note any narrative inconsistencies

### After Writing
1. Read through for flow
2. Check character consistency
3. Verify plot logic
4. Confirm page length (~45-50 pages)
5. Submit for peer review

---

## Review & Revision Process

### Draft Stages
1. **First Draft** - Initial screenplay
2. **Network Review** - Studio notes
3. **Second Draft** - Incorporate feedback
4. **Production Draft** - Final version for set
5. **Shooting Draft** - Final with any on-set revisions

### Approval Chain
- Writer → Producer → Network → Production
